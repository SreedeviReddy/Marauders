GUIDE

Attendance: One can check their attendance from which date to which by entering the appropriate values in the box provided.
Leave: One can apply for leave and also check the status if the leave has been granted or not.
Notifications: One can check the pproject updates as well as any new poll or survey.
Logout: To logout, go to the settings option and then select logout.
Profile: To check and edit your profile, go to the settings page and select profile and make the necessary changes.
